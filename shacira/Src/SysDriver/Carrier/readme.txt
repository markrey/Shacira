
   This directory contains source for a Carrier I/O driver
   running under Windows NT 4 (WINNT) or Windows 2000 / Windows XP
   (WIN2000). To generate the driver install the specific DDK
   (Windows NT DDK for Windows NT, Windows 2000 DDK for Windows 2000
    and XP). Start the Free or checked build environment.
   Change into the driver directory (WINNT or WIN2000 respectively).
   Execute "build".
